* attributes
    * strength, dex, constitution, intelligence, wisdom, charasma
    * need starting values per class, level up values per class
    * need to know how these values impact health, energy, focus
    * str
        * impacts melee attack (hand to hand or weapon) power
        * impacts melee attack hit chance
        * impacts skill checks for things like breaking down doors, feats of str
    * dex
        * impacts initiative
        * impacts ranged attack
        * impacts dodge
        * impacts stealth
        * impacts skill checks for things like lockpicking
    * con
        * impacts health
        * impacts ac
        * impacts resistance to status effects
        * impacts recovery time
    * int
        * impacts options/perceptions in puzzles
        * impacts spell power
        * impacts skill checks for things like knowledge, lore, information
        * impacts crafting of items
    * wis
        * impacts focus
        * impacts negotiation (detect lies, assess motives)
        * impacts skill checks related to tracking, foraging, understanding the environment
    * charasma
        * impacts negotiation and deescalation
        * impacts morale in combat, giving boosts to others skills/attributes
* classes
    * technologist
        * magic user
        * primary int, wis
        * crafting
        * puzzle solving
        * can create traps or gadgets unavailable to other classes
    * survivalist
        * physical class
        * resource gathering, longer expiditions
        * primary str, dex
        * can access areas unavailable to other classes through nimbleness
    * negotiator
        * bard type
        * primary wis, cha
        * resolve conflict
        * leverage npc for resources or hints
        * persuasion and insight allows access to hidden resources, unique interaction outcomes
